```
Usage: sorter [OPTION]... [FILE]  
Sort numbers, words or lines.  
  
Mandatory arguments to long options are mandatory for short options too.  
   -d, -dataType [long|word|line]     chose the appropriate data type  
   -s, -sortingType [natural|byCount] specify natural sorting order or by count  
   -f, -inputFile [FILE]   file to read from  
   -o, -outputFile [FILE]  file to write to, with no FILE write to standard output  
```
